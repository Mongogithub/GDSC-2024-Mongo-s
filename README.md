## Getting Started

### Dependencies
Before beginning you need to have
* Git
* Node.JS

### Installing

```
git clone git@github.com:vichekaoeun/GDSC-2024.git
```
or with SSH
```
git clone git@github.com:vichekaoeun/GDSC-2024.git
```
Make sure to run *npm install* to get all packages
```
npm install
```
### Executing program

* How to run the program
* Development:
```
npm run dev
```
* Production
```
npm run build
```
